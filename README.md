## Dataset

The dataset contains pizza sales information such as:

* Orders and order details
* Pizza types, sizes, and prices

These tables are used to simulate a real-world sales database.

---

## Analysis Goals

Using SQL, this project answers questions like:

* Total orders and total revenue
* Best-selling pizzas
* Sales by pizza category and size
* Order trends over time

---

## SQL Concepts Used

* SELECT, WHERE, ORDER BY
* GROUP BY with aggregates (SUM, COUNT)
* JOINs
* Subqueries
* Date functions

---

## How to Run

1. Extract `pizza_sales.zip`
2. Import CSV files into a SQL database
3. Run `project.sql`
